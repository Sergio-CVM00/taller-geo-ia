La cara visible de GEOIA: tu mapa en el navegador.

MÓDULO 2 · DÍA 2

El backend

La lógica de GEOIA: una API que sirve los lugares.

  • Un servidor con Python + FastAPI.
  • GET /lugares entrega la lista; POST /lugares añade uno.
  • Se prueba solo, en /docs — sin necesitar el mapa.
🎯 Una API viva en localhost, con su documentación automática.
